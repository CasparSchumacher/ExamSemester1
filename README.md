# Casomorphin

Inoffizielles, privates Lernprojekt zum Üben von Übungsfragen, geordnet nach **Modul → Modulwoche → Lernziel**.

- `index.html` — Startseite: Semester 1–10 (S1 verfügbar, 2–10 in Vorbereitung)
- `semester-1.html` — Klausurtrainer Semester 1, Reiter: Üben · Quiz · Lernziele · ★ Markiert · Hilfe
  - Klausurmodus, Filter (Modul/Modulwoche), Suche, Lesezeichen (lokal), eingebettete Abbildungen
  - Feedback/Fehlermeldung über ein Formular (Relay, keine Adresse im Quelltext)
  - Daten: `data/semester-1.js`, Bilder: `img/`
- `Abbildungen.html` — Abbildungs-Galerie

**Haftungsausschluss:** Keine Verbindung zu einer Universität/Fakultät/Fachschaft. Die Fragen orientieren
sich an Lernzielen und könnten so oder ähnlich in Klausuren vorkommen; keine offiziellen Prüfungsfragen.
Keine Gewähr für Richtigkeit, Vollständigkeit oder Aktualität. Nutzung auf eigene Verantwortung.

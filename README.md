# ExamSemester1 — Altklausuren-Lernkartei (Charité MSM, 1. Semester)

Interaktive Lernkartei + Abbildungs-Galerie aus 25 Altklausuren & Gedächtnisprotokollen,
zugeordnet zu den Lernzielen der Module **M02 + M03 + M04**.

**Webseite (einfach im Browser öffnen):** https://casparschumacher.github.io/ExamSemester1/

- `index.html` — Startseite
- `Lernkartei_Gesamt.html` — Lernkartei (Modi: Lernen / Quiz / Frei, Suche, eingebettete Abbildungen)
- `Abbildungen.html` — durchsuchbare Abbildungs-Galerie

Erstellt zur Prüfungsvorbereitung. Ohne Gewähr auf Richtigkeit aller Lösungen.

# ExamSemester1 — Altklausuren-Klausurtrainer (Charité MSM)

Web-App zum Üben von Altklausurfragen, geordnet nach **Modul → Modulwoche → Lernziel**.

**Webseite (einfach im Browser öffnen):** https://casparschumacher.github.io/ExamSemester1/

- `index.html` — Startseite/Hub: alle Semester 1–10 (Semester 1 verfügbar, 2–10 in Vorbereitung)
- `semester-1.html` — Klausurtrainer Semester 1 (Module M02/M03/M04)
  - **Klausurmodus:** Antwort anklicken → richtig/falsch + Punktestand
  - Filter nach **Modul** und **Modulwoche**, Volltextsuche
  - klare Zuordnung jeder Frage zu **Modul · Modulwoche · Lernziel** (zum Nachlernen)
  - eingebettete **Abbildungen** (Klick = vergrößern)
  - PDF: im Browser *Drucken → Als PDF sichern* (zeigt alle Lösungen)
- `Abbildungen.html` — durchsuchbare Abbildungs-Galerie

Modulangaben für Semester 2–10 sind eine Vorschau gem. Fachschaft (ohne Gewähr).
Erstellt zur Prüfungsvorbereitung; ohne Gewähr auf Richtigkeit aller Lösungen.

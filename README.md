# ExamSemester1 — Altklausuren-Klausurtrainer (Charité MSM)

Web-App zum Üben von Altklausurfragen, geordnet nach **Modul → Modulwoche → Lernziel**.

**Webseite (im Browser öffnen):** https://casparschumacher.github.io/ExamSemester1/

- `index.html` — Startseite/Hub: Semester 1–10 (S1 verfügbar, 2–10 in Vorbereitung)
- `semester-1.html` — Klausurtrainer Semester 1 (Module M02/M03/M04), Reiter:
  - **Üben** — alle Fragen, Filter nach Modul/Modulwoche + Suche (Klausurmodus)
  - **Quiz** — Module/Modulwochen wählen, Anzahl (10/20/40/alle/eigene) → zufälliges Set,
    eine Frage nach der anderen, Auswertung am Ende
  - **Lernziele** — alle Lernziele je Modul/Modulwoche; Klick → zugehörige Fragen
  - **★ Markiert** — schwierige Fragen sammeln (im Browser gespeichert), wiederholen
  - **Hilfe** — Anleitung + Fehler/Feedback per E-Mail
  - pro Frage: **☆ markieren** und **⚑ Fehler melden** (E-Mail an Support)
  - Abbildungen eingebettet (Klick = vergrößern); PDF via Drucken → Als PDF sichern
  - `data/semester-1.js` (Fragen+Lernziele), `img/` (Abbildungen)
- `Abbildungen.html` — durchsuchbare Abbildungs-Galerie

Modulangaben für Semester 2–10 sind eine Vorschau gem. Fachschaft (ohne Gewähr).
Erstellt zur Prüfungsvorbereitung; ohne Gewähr auf Richtigkeit aller Lösungen.
